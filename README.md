# HackNTell 

HackNTell is a simple project built with Javascript!

## Requirements

* [NodeJS](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/lang/en/docs/install/)

## Development

Set up the project with

```
$ make install
```

To start the server

```
$ make up
```

Lint the source code

```
$ make lint
```
